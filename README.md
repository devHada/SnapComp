## What is SnapComponent?

SnapComponent is an AI-powered React component generator. Type what you want, get the JSX code and a live preview — instantly.

No backend. No setup. Just describe and generate.

## Features

- 💬 **Chat Interface** — describe your component in plain English
- ⚡ **Instant Generation** — powered by Groq API + llama-3.3-70b-versatile
- 👁️ **Live Preview** — see your component render in real time
- 📋 **Code View** — copy the generated JSX with one click
- 🌙 **Dark / Light Mode** — full theme support
- 📱 **Responsive** — works on mobile too

## Tech Stack

- React + Vite
- Tailwind CSS v4
- Framer Motion
- Groq API (llama-3.3-70b-versatile)
- Axios

## Getting Started

```bash
git clone https://github.com/argenis972/snap-component
cd snap-component
npm install
```

Create a `.env` file:

```
VITE_GROQ_API_KEY=your_groq_api_key
```

```bash
npm run dev
```

## Usage

1. Type a component description in the chat
2. Hit Enter or click send
3. See the live preview on the right
4. Switch to Code tab to copy the JSX

## Examples

- _"Create a login form with dark background and orange accents"_
- _"Make an animated counter that counts from 0 to 100"_
- _"Build a todo list where you can add and delete items"_
